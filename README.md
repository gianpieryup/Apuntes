## 💻 Visual Studio Code (VS Code)

### ⌨️ Atajos Esenciales
*   **Gestión de Líneas:**
    *   Comentar: `Ctrl` + `K` + `C`
    *   Descomentar: `Ctrl` + `K` + `U`
*   **Visualización:**
    *   Ajuste de texto (**Word Wrap**): `Alt` + `Z` (Nota: En tu versión decía `Ctrl`, pero el estándar es `Alt`).
    *   Zoom: `Ctrl` + `+` / `-` (Resetear con `Ctrl` + `0`).
*   **Navegación:**
    *   Configuraciones: `Ctrl + ,`
    *   Terminal (Mostrar/Ocultar): `Ctrl + j`

### ⚙️ Configuración Recomendada (Settings JSON)
Para activar estas funciones, busca el nombre en `Ctrl + ,` y marca la casilla:
*   **Mouse Wheel Zoom:** Permite hacer zoom con la ruedita del ratón.
*   **Wrap Tabs:** Si tienes muchos archivos abiertos, los muestra en varias filas en lugar de una barra desplazable.
*   **Pin Tabs:** `Click derecho > Pin` para fijar archivos importantes.

### 🔌 Extensiones Sugeridas
*   🐳 **Docker & Dev Containers  (Microsoft):** Imprescindibles para trabajar con contenedores.
*   🎨 **vscode-icons  (Microsoft):** Mejora la identificación visual de archivos.
*   ✨ **Prettier:** (Recomendación Senior) Para formatear JSON, JS, HTML y CSS automáticamente.

## 🛠️ Entorno de Desarrollo (Terminal & SO)

### 🐚 Terminal Preferida: Git Bash
Se recomienda usar **Git Bash** dentro de VS Code para aprovechar los comandos nativos de Linux (`ls`, `mkdir -p`, `curl`, `grep`).

> **Tip Senior:** En Windows 11, usa **Windows Terminal** para gestionar múltiples pestañas de PowerShell y Bash con mejor rendimiento.

### 📂 Estructura de Archivos en Windows
Es fundamental saber dónde se instalan tus herramientas para configurar variables de entorno (PATH):
*   `C:\Program Files`: Aplicaciones de **64 bits**.
*   `C:\Program Files (x86)`: Aplicaciones de **32 bits**.

## 💡 Tips de Programación y Hardware

### ⌨️ Trucos de Teclado

- No te funciona el click derecha del pad de la laptod : Presiona con *dos dedos* al mismo tiempo :)
- Modo del cursor: Que sea un palito (<kbd>**Fn**</kbd> + <kbd>[Ins| ImpPrint]</kbd>)
- Sacar carácter <kbd>`</kbd> : Presionar dos veces la Combinacion (<kbd>**Alt Gr**</kbd> + <kbd>}</kbd>)
- Tomar Screenchot (<kbd>**Windows**</kbd> + <kbd>Impr Pant</kbd>). Esto se guarda en la carpeta de Imagenes
- Tomar captura de pantalla (<kbd>**Shift**</kbd> + <kbd>**Windows**</kbd> + <kbd>S</kbd>) se guarda en el portapapeles
- Abrir la terminal cmd desde cualquier folder: Escribe `cmd` en la ruta donde estas parado y presiona enter
- En Windows 11, puedes descargar el programa ``Terminal`` que es un gestor de consolas con personalizacion. Yo lo tengo y se ve mas bonito. Para aumentarle cosas como la mostrar en que rama de GIT estoy en el promt, seguir las intrucciones de un video de youtube `How to Display Git Branch Name in PowerShell Prompt`




### 🧠 Memotecnia de Lógica (Fe de erratas)
